# GAN_Project_DL — Simple GAN & AttGAN on CelebA

Implementation of two GAN models on CelebA, progressing from an unconditional baseline to a conditional attribute-editing network.

| Model | Paper | Task |
|---|---|---|
| **Simple GAN** | Radford et al., 2015 (DCGAN) | Unconditional face generation |
| **AttGAN** | He et al., 2019 | Facial attribute editing |

## Experiments

| Experiment | λ_rec | λ_cls_D | λ_cls_G | Hypothesis |
|---|---|---|---|---|
| `exp1_baseline` | 100 | 10 | 1 | Paper defaults — balanced |
| `exp2_high_rec` | 200 | 10 | 1 | Stronger identity preservation, softer edits |
| `exp3_strong_attr` | 50 | 10 | 5 | Sharper edits, some identity loss |

Each config is a subclass of `Config` in `experiments/` that only overrides the changed values. Results are isolated in `results/<experiment_name>/`.

## Metrics

### FID (Fréchet Inception Distance)
Standard GAN metric (Heusel et al., 2017). Fréchet distance between multivariate Gaussians fitted to Inception-v3 pool-3 features of real and generated images. Lower is better.

### DACID (Dany Aissa & Clara's Image Distance)
Custom lightweight metric. L2 distance between **mean** Inception feature vectors — no covariance estimation, so faster and easier to interpret. Lower is better.

Both run automatically after training (toggle with `--no-metrics`). Implemented in `src/metrics.py`.

## Architecture overview

### Simple GAN (DCGAN)
```
z (100×1×1) → Generator (5× ConvTranspose2d) → image (3×64×64)
                                                       │
                                             Discriminator (5× Conv2d)
                                                       │
                                                real/fake scalar
Loss G: BCE(D(G(z)), 1)
Loss D: BCE(D(x), 1) + BCE(D(G(z)), 0)
```

### AttGAN
```
image → Encoder → z + target_attrs (tiled) → Generator → fake_image
                                                               │
                                             Discriminator ←──┘
                                               ├─ adv_head (LSGAN)
                                               └─ cls_head (BCE, 13 attrs)
```

| Loss | λ default | Purpose |
|---|---|---|
| L_adv (MSE/LSGAN) | — | Realism |
| L_cls (BCE) | D=10 / G=1 | Attribute accuracy |
| L_rec (L1) | 100 | Identity preservation |
